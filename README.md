# 货运APP、货运小程序

货运、货拉拉、货运平台、网络货运、货运小程序、货运搬家、货运APP、同城配送、搬家小程序、搬家预约、物流运输、TMS系统、物流运输、物流软件、TMS运输管理系统、物流管理系统、物流系统、物流搬运、物流配送

# 技术栈

后台服务 springboot+mybatisplus+mysql
用户端 uniapp（vue语法）
管理后台 vue+elementUi
